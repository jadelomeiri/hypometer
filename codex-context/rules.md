# Codex Rules

When editing this project:

- Preserve existing styling
- Preserve existing layout
- Preserve existing behaviour
- Avoid redesign
- Avoid UI complexity
- Avoid feature bloat
- Prefer minimal changes
- Prefer simple solutions

If there is a conflict between:

adding features
vs
keeping simplicity

Choose simplicity.

If there is a conflict between:

new design
vs
existing design

Keep existing design.

If unsure, ask for clarification instead of redesigning.

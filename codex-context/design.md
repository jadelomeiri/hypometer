# HypeOmeter Design Rules

The current styling and colours are the source of truth.

Do NOT change the design unless explicitly asked.

Rules:

- Keep the same colours
- Keep the same typography
- Keep the same spacing
- Keep the same layout
- Keep the same component style

Avoid:

- redesigning cards
- changing fonts
- changing theme
- adding gradients
- adding new color palettes
- adding heavy shadows
- adding flashy UI
- adding animations unless asked

Design should feel:

- minimal
- clean
- professional
- fast
- focused

Not:

- noisy
- crowded
- overdesigned

If a new feature requires UI:

Prefer the simplest possible UI.

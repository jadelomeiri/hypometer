# HypeOmeter Product Context

HypeOmeter is a simple web app that analyzes AI-related posts and scores them for hype vs substance.

Core flow:

1. User pastes a post
2. User clicks analyze
3. App returns scores and explanation

The product must stay simple.

Important principles:

- The app should feel fast and minimal
- The UI must not become cluttered
- Avoid feature bloat
- Avoid unnecessary controls
- Avoid redesigning layout unless explicitly requested

The original version of the app is the reference for simplicity.

All new features must preserve the original feel.

Allowed additions:
- Share link
- Public result page
- Share text
- OG image
- Subtle attribution
- Small footer
- Minimal CTA

Not allowed unless explicitly requested:
- new modes
- complex filters
- tabs
- dashboards
- redesigns
- big banners
- noisy UI
